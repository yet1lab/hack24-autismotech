Termos de Uso do Plugin Traduz

1. Introdução

Bem-vindo(a)! Este plugin le a estrutura do site e adapta ele para funcionar com leitores de tela. Sempre que necessario utiliza inteligência artificial para ler uma imagem e gerar uma descrição da imagem, fornecendo uma experiência inclusiva e personalizada para todos os usuários, inicialmente pessoas de baixa visão e usuários de leitores de tela. Ao utilizar o plugin, você concorda com os Termos de Uso e nossa Política de Privacidade.

2. Coleta e Tratamento de Dados Pessoais

Em conformidade com a Lei Geral de Proteção de Dados (LGPD), informamos que:

Quais dados coletamos?


Podemos coletar informações como dados de navegação (cookies), preferências de acessibilidade ajustadas no plugin e eventuais interações necessárias para personalização da experiência.

Como utilizamos os dados?


Os dados são utilizados exclusivamente para:
  • Melhorar a experiência do usuário.
  • Personalizar os ajustes de acessibilidade.
  • Realizar análises anônimas para otimização do sistema.

3. Inteligência Artificial e Funcionalidade

Este plugin utiliza IA para:

  • Ler imagens e adicionar sua descrição
  • Futuramente adicionar tradução de textos para autistas

4. Consentimento

O uso deste plugin requer seu consentimento para processar os dados mencionados. Você pode optar por não permitir o uso do plugin ou retirar seu consentimento a qualquer momento, desativando o plugin ou excluindo cookies relacionados.

5. Segurança dos Dados

Tomamos medidas rigorosas para proteger seus dados contra acesso não autorizado, perda ou alterações indevidas. Os dados coletados são armazenados em servidores seguros, com criptografia e acesso restrito.

Conforme a LGPD, você tem os seguintes direitos:

  • Solicitar acesso, correção ou exclusão de dados pessoais.
  • Saber como seus dados estão sendo tratados.

O plugin é uma ferramenta auxiliar. Não garantimos que todas as barreiras de acessibilidade serão eliminadas. A responsabilidade pela acessibilidade integral do conteúdo é do proprietário do site onde o plugin é utilizado.
